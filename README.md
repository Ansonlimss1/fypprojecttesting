# MyCar Advisor Malaysia

This project is a Streamlit-based car advisory chatbot for Malaysian car owners. It is suitable for an FYP demo and includes:

- Greetings function
- Car information function
- Why-this-car explanation
- Car comparison
- Driving usage recommendations
- Seat-based recommendations
- Maintenance advice

## Run locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the Streamlit app:

```bash
streamlit run projecttesting.py
```

3. Streamlit will open a local URL such as `http://localhost:8501`.

## Make it public

1. Push this folder to a GitHub repository.
2. Go to Streamlit Community Cloud.
3. Create a new app and connect your GitHub repository.
4. Set the main file path to `projecttesting.py`.
5. Deploy the app.
6. Streamlit will provide a public link that you can share with other users.
